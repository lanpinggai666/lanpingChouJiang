# 蓝屏抽奖机
**一个置顶在屏幕左侧中间方便老师在上课时点名的程序（你用来抽奖也行）。是鄙人的第一个作品。<br>
使用WPF编写，拥有（伪）Fluent UI风格。运行需要.NET 8.0运行库。（虽然不知道这么简单的程序要运行库，但我真的不会其<br>
他的啊😭😭😭）**
